Основа исследования

Проведено исследование как дыхательный сигнал влияет на работу сердца через вегетативную нервную систему. 
Для этого было рассмотрено три разных состояния: нормальное состояние человека а также 
после препаратов атропина и пропранолола.

Как исследовали

Инструменты:
1. Источник - https://engee.com/community/cn/catalogs/projects/chastotnyi-analiz-vegetativnogo-kontrolia-serdechnogo-ritma
2. Язык Python
3. Среда разработки Google Colab
4. Библиотеки scipy matplotlib

Сначала была создана функция freq_analysis которая выполняет частотный анализ 
передаточной функции от сигнала дыхания к сигналу частоты сердечных сокращений. 
В этой функции применяется оконная функция Ханна выполняется быстрое преобразование 
Фурье и рассчитываются амплитудно частотная и фазочастотная характеристики.

Далее была реализована простая модель генерации физиологических сигналов. 
Для каждого из трех сценариев генерировался дыхательный сигнал с помощью 
линейно частотно модулированного сигнала. На основе этого сигнала рассчитывались 
изменения частоты сердечных сокращений и артериального давления.

Для визуализации результатов были созданы функции построения графиков. 
Для каждого сценария выводились временные графики объема легких частоты 
сердечных сокращений и артериального давления. Также были построены графики 
частотных характеристик для каждого состояния и итоговый сравнительный график.

Дыхательный сигнал генерировался с помощью линейно частотно модулированного сигнала. 
Для каждого сценария рассчитывалась передаточная функция от объема легких к частоте 
сердечных сокращений. 

Также были построены временные графики физиологических показателей и частотные 
характеристики амплитудно частотная и фазочастотная.


Результаты исследования

В нормальном состоянии наблюдается выраженная связь между дыханием и изменениями 
частоты сердечных сокращений. 

При введении атропина происходит ослабление реакции на высоких частотах и усиление 
на низких частотах. 

При введении пропранолола преобладает быстрый парасимпатический отклик с малым 
фазовым сдвигом.

Частотный анализ позволил наглядно показать различия в динамике работы вегетативной 
нервной системы в разных условиях.# -
